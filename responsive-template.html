YA(c)F - Yet Another (CSS) Framework

YA(c)F (Yet Another (CSS) Framework) is a flexible, responsive, lightweight CSS framework that gives you everything you need to build robust, scalable websites without a lot of extraneous or confusing markup. This page serves to demonstrate the various features provided by YAF.

  • CSS Grid provides responsive, pixel-perfect positioning and makes the most demanding layouts easy to render.
  • Theme files in SASS format makes site-wide customization a snap.

Responsive Grid (CSS Grid Style)

This page serves to demonstrate a responsive grid system in 12- and 16-column flavors. Using CSS grid and media queries, the page reflows and serves screens from 240px wide to full-size desktop resolutions.

Unlike my earlier fluid grid, IE 7 and 8 will not be served a separate stylesheet; their current browser share is too small to be catered to. This has been tested and works as expected in all major browsers with 2% or greater market share as of February 2018.
